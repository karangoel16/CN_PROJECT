To run this project (Client/server) , we would need to perform certain steps:-
1. Run the server on (localhost/Server with IP address).
2. Change the IP address of the server in the Client module.
3. Compile the code.
4. Run the server on localhost or server wherever you wish to.
5. Run the program.
6. Enter the location where you would like to store the file(e.g. C:\Localhost\test1\)
7. You are good to go and run the various functionalities of the program.
